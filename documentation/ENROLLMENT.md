### ALL /api/v1/auth/bulk-import/api/v1/auth/bulk-import
**Method:** ALL
**Endpoint:** /api/v1/auth/bulk-import/api/v1/auth/bulk-import
**Description:** Controller to handle bulk user registration via CSV upload. This endpoint is secured and accessible only to administrative roles. <p> To receive real-time SSE progress updates, the client must: </p> <ol> <li>Generate a unique {@code sessionId} (UUID).</li> <li>Subscribe to {@code GET /api/v1/bulk-import/stream/{sessionId}} first.</li> <li>Then post the CSV here with the {@code X-Session-Id: <sessionId>} header.</li> </ol> <p> If {@code X-Session-Id} is omitted, the import still runs — just without SSE updates. </p>

**Security Requirement:**
- `Authority / Scope`: PermitAll / Unspecified
- `Class Context`: BulkImportController

- `None (Public Endpoint)`
**Query Params ->**
None
**body ->**
None
**Request DTO ->** None
**Response DTO ->** `ResponseEntity` wrapping `StandardResponse` { success: `Boolean`, message: `String`, timestamp: `String` }

### ALL /api/v1/bulk-import/stream/api/v1/bulk-import/stream
**Method:** ALL
**Endpoint:** /api/v1/bulk-import/stream/api/v1/bulk-import/stream
**Description:** SSE endpoint that the frontend subscribes to BEFORE triggering a bulk import. <p><b>Usage flow:</b></p> <ol> <li>Frontend generates a unique {@code sessionId} (e.g. UUID v4).</li> <li>Frontend opens an EventSource to {@code GET /api/v1/bulk-import/stream/{sessionId}}.</li> <li>Frontend uploads the CSV to {@code POST /api/v1/auth/bulk-import/{userType}} with the same {@code sessionId} header.</li> <li>Backend emits {@code ROW_SUCCESS}, {@code ROW_FAILURE} per row, then {@code JOB_COMPLETE} or {@code JOB_FAILED} at the end.</li> </ol> <p>The stream endpoint is <b>intentionally unauthenticated at the HTTP layer</b> because {@code EventSource} in browsers cannot set custom headers (e.g. Bearer token). The sessionId acts as a single-use opaque token — it is unguessable (UUID) and is only useful for the duration of one import job.</p>

**Security Requirement:**
- `Authority / Scope`: PermitAll / Unspecified
- `Class Context`: BulkImportSseController
**Headers ->**
- `None (Public Endpoint)`
**Query Params ->**
None
**body ->**
None
**Request DTO ->** None
**Response DTO ->** `ResponseEntity` wrapping `StandardResponse` { success: `Boolean`, message: `String`, timestamp: `String` }
